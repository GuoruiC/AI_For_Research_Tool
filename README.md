# AI For Research Tool

提升科研效率的 AI 工具与小技巧

内容覆盖论文检索、创新性分析、代码辅助、论文润色，以及全部工具的官网直达索引，适合用于实验室分享、组会展示或个人科研工具导航。

## 项目特点

- 本地静态站点，无需后端服务
- 首屏为封面页，后续内容按幻灯片逐页切换
- 整体视觉以白底黑字为主，辅以科技感蓝绿高亮
- 文中主要工具均附官网链接
- 部分工具使用本地化 logo 资源，缺失 logo 的工具使用统一风格图标替代
- 支持键盘、按钮和鼠标滚轮切换页面

## 目录结构

```text
AI_For_Research_Tool/
├─ assets/
│  └─ logos/        # 本地保存的工具图标资源
├─ app.js           # 幻灯片切换逻辑
├─ index.html       # 页面结构与内容
├─ styles.css       # 页面样式
└─ README.md
```

## 页面内容

当前页面共 8 页：

1. 封面
2. Workflow 总览
3. 找论文与现状调研
4. 分析论文动机与创新性
5. 代码辅助撰写与实验实现
6. 论文修改与润色
7. 通用 Prompt 技巧与注意事项
8. 全部工具官网直达索引

## 使用方式

直接在浏览器中打开 `index.html` 即可。

支持的切换方式：

- 点击底部左右切换按钮
- 使用键盘 `←` / `→`
- 使用 `PageUp` / `PageDown`
- 使用空格键切换下一页
- 使用鼠标滚轮切换页面

## 适用场景

- 实验室内部分享
- 组会展示
- 科研工具导航页
- 个人知识库入口页
